# CHREMATOS
The human race fell almost 3000 years ago, to weapons of their own hubris. Little else survived. Mammals, avians, reptiles, and the like all died out from radiation poisoning.
But the bugs were able to take what was left and build up from there.
---
Built with UNITY 2D engine



yes the allegories are VERY heavy handed, what are you going to do about it?

## Running the game
---
- On Windows: Download the latest release, extract the ZIP file, and run the file "Bug yuri game.exe"
- Other: download the source code, open in unity, go to build setting, build on platform of choice.
